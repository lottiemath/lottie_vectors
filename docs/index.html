Lottie Vectors

Welcome to the website for Lottie Vectors a program that works with Mathlab(r) to display 2 dimensional vectors within a 3 dimensional window. You can load in vectors in a simple format from the command line, the workspace or a text file.



LATEST

Lottie Vectors is currently in Version 1.14 which has a bug in the ‘load from workspace’ window . This will be fixed in 1.15 as well as a feature to add vectors from the window.

A new version of the user manual is being written which is updated for 1.15.

You can download the latest versions of the program and manual from the root repository of the github github.com/lottiemath/lottie_vectors/ just look for the latest version numbers at the end of the filename.
